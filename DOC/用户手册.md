## 1. 系统概述

MediCare 是一款面向社区医院、专科门诊、乡镇卫生院的桌面级智慧医疗门诊管理系统。系统覆盖患者管理、挂号预约、医生工作站、病历管理、药品库存、处方管理六大核心模块，支持单机或局域网部署。

**核心功能一览：**

| 模块 | 功能 |
|------|------|
| 患者管理 | 患者登记、档案查询、编辑、删除 |
| 挂号预约 | 科室选择、排班查看、现场挂号、叫号/完成/取消 |
| 医生工作站 | 今日候诊、患者叫号、病历书写、完成就诊 |
| 病历管理 | 全量病历检索、详情查看、按患者/医生/诊断搜索 |
| 药品库存 | 入库/出库、库存查询、有效期预警、库存预警 |
| 处方管理 | 处方开立、药品检索、明细管理、事务级库存扣减 |
| 基础数据 | 科室管理、医生管理、排班管理 |
| 系统设置 | 管理员管理、密码修改 |

## 2. 安装与启动

### 2.1 环境要求

- **JDK**：OpenJDK 17+
- **数据库**：MySQL 8.0
- **内存**：≥ 4GB
- **显示器**：≥ 1280×720

### 2.2 数据库初始化

```bash
mysql -u root -p < sql/schema_baseline.sql
mysql -u root -p < sql/seed_data.sql
```

### 2.3 启动系统

```bash
java -jar mediCare-1.0.0-SNAPSHOT.jar
```

## 4. 患者管理

### 4.1 功能说明

患者管理模块用于维护患者档案，支持新增、编辑、删除患者信息，以及通过姓名/手机号/身份证号搜索患者。

### 4.2 操作步骤

#### 4.2.1 查看患者列表

1. 点击左侧导航【患者管理】
2. 右侧显示患者列表表格

1. 点击顶部【新建档案】按钮
2. 在弹出的患者编辑对话框中填写信息：
   - **身份证号***（必填）
   - **姓名***（必填）
   - **性别**：下拉选择（女性/男性），默认男性
   - **出生日期**：日期选择器

3. 在弹出的对话框中修改信息

## 5. 挂号预约

### 5.1 功能说明

挂号预约模块用于查看医生排班号源，选择号源后为患者办理现场挂号，支持叫号、完成就诊、取消挂号等操作。

### 5.2 操作步骤

#### 5.2.1 查看号源

1. 点击左侧导航【挂号预约】
2. 顶部选择就诊日期和科室（可选）
3. 点击【查询号源】按钮
4. 左侧显示可用号源列表

> 【截图位置 6：挂号预约界面】
>
> 请截取挂号预约模块的完整界面截图。
> 截图应包含：顶部日期/科室筛选、左侧号源表格、右侧挂号记录。

#### 5.2.2 现场挂号

1. 在左侧号源列表中选中一条号源（剩余号源 > 0）
2. 点击右下角【挂号】按钮
3. 在弹出的【选择患者】对话框中：
   - 在搜索框输入患者姓名进行模糊查询
   - 或直接从列表中点选患者
   - 如患者不在列表中，点击【新增患者】按钮建档
4. 选中患者后点击【确定挂号】
5. 系统提示挂号成功，并显示挂号序号

> 【截图位置 7：患者选择对话框】
>
> 请截取点击【挂号】后弹出的患者选择对话框。
> 截图应包含：搜索栏、患者列表、新增患者按钮、确定/取消按钮。

#### 5.2.3 叫号

1. 在右侧【今日挂号记录】中选中一条候诊记录
2. 点击右下角【叫号】按钮
3. 患者状态从"候诊"变为"就诊中"

#### 5.2.4 完成就诊

1. 在右侧挂号记录中选中一条"就诊中"记录
2. 点击【完成】按钮
3. 患者状态变为"已完成"

#### 5.2.5 取消挂号

1. 在右侧挂号记录中选中一条记录
2. 点击【取消】按钮
3. 在确认对话框中点击【确定】
4. 号源自动释放，可再次挂号

#### 5.2.6 筛选挂号记录

点击右侧上方的筛选按钮（全部/候诊/就诊中/已完成）可快速筛选不同状态的挂号记录。

## 6. 医生工作站

### 6.1 功能说明

医生工作站供医生查看今日候诊患者列表，进行叫号操作，并为患者书写结构化病历。

### 6.2 操作步骤

#### 6.2.1 选择医生

1. 点击左侧导航【医生工作站】
2. 在顶部下拉框中选择当前出诊医生
3. 左侧自动加载该医生的今日候诊列表

> 【截图位置 8：医生工作站界面】
>
> 请截取医生工作站模块的完整界面截图。

#### 6.2.2 叫号

1. 在左侧候诊列表中选中一位患者
2. 点击【叫号】按钮

1. 选中患者后，系统自动加载该患者的历史病历（如有）

   - **主诉***（必填）
   - **现病史**
   - **既往史**
   - **体格检查**
   - **诊断***（必填）
   - **建议**
3. 点击【保存病历】按钮
4. 系统自动将病历关联到当前挂号记录

#### 6.2.4 完成就诊

1. 病历保存后，点击【完成就诊】按钮
2. 该患者从候诊列表中移除（或状态更新）

#### 6.2.5 查看历史病历

选中患者后，右侧自动显示该患者的历史病历时间轴，可查看既往就诊记录。

## 8. 药品库存

### 8.1 功能说明

药品库存模块用于管理药品档案，支持入库、出库操作，实时显示库存预警和有效期预警。

### 8.2 操作步骤

#### 8.2.1 查看药品列表

1. 点击左侧导航【药品库存】
2. 显示药品库存列表，顶部显示统计信息：
   - 药品总数
   - 库存预警数量（红色）
   - 近效期预警数量（橙色）

> 【截图位置 12：药品库存界面】

1. 点击顶部【新增药品】按钮
2. 在弹出的药品编辑对话框中填写：
   - **药品名称***
   - **规格**
   - **单位**
   - **零售价**
   - **安全库存**
   - **批号**
   - **有效期**
   - **拼音码**
   - **生产厂家**
   - **状态**（启用/停用）

> 【截图位置 13：药品编辑弹窗】
>
> 请截取药品新增/编辑弹窗的截图。

#### 8.2.3 编辑药品

1. 在药品列表中找到目标药品
2. 点击【编辑】按钮
3. 修改信息后保存

#### 8.2.4 删除药品

1. 在药品列表中找到目标药品
2. 点击【删除】按钮
3. 确认删除（注意：有关联库存记录的药品需先删除记录）

#### 8.2.5 药品入库

1. 在药品列表中找到目标药品
2. 点击【出入库】按钮
3. 在弹出的出入库对话框中：
   - 选择操作类型：**入库**
   - 输入入库数量
   - 输入批号（如与当前不同则更新药品批号）
   - 输入有效期（如与当前不同则更新药品有效期）
   - 输入操作人
   - 输入备注
4. 点击【执行】

> 【截图位置 14：出入库弹窗】
>
> 请截取出入库操作弹窗的截图，显示入库状态下的字段。

#### 8.2.6 药品出库

1. 在药品列表中找到目标药品
2. 点击【出入库】按钮
3. 选择操作类型：**出库**
4. 输入出库数量（不能超过当前库存）
5. 输入操作人、备注
6. 点击【执行】

#### 8.2.7 搜索药品

在顶部搜索框输入药品名称或拼音码，点击【查询】按钮。

3. 系统提示"处方开立成功"

## 11. 系统设置

### 11.1 功能说明

系统设置模块用于管理管理员账号和修改当前用户密码。

### 11.2 操作步骤

#### 11.2.1 管理员管理

1. 点击左侧导航【系统设置】
2. 默认显示管理员列表

## 12. 常见问题

### Q1：登录时提示"用户名或密码错误"

- 确认输入的用户名和密码正确（默认：admin / admin123）
- 检查大小写是否一致
- 如忘记密码，需联系系统管理员重置

### Q2：挂号时提示"号源已售罄"

- 该时段号源已挂完，请选择其他时段或医生
- 如有患者取消挂号，号源会自动释放

### Q3：处方保存失败，提示"库存不足"

- 某药品的当前库存不足以满足处方中的数量
- 请减少该药品数量，或先到【药品库存】模块进行入库操作

### Q4：删除药品时提示"外键约束失败"

- 该药品有关联的库存记录（inventory_log）
- 系统设计上禁止级联删除，需先删除库存记录

### Q5：界面显示乱码

- 确认数据库连接串中包含 `characterEncoding=UTF-8`
- 确认数据库字符集为 `utf8mb4`
