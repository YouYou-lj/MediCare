## 二、数据库部署

### 2.1 安装 MySQL 8.0

```bash
# macOS (Homebrew)
brew install mysql@8.0
brew services start mysql

# Windows / Linux
# 请从官网下载安装包或使用包管理器安装
```

### 2.2 创建数据库用户

```sql
-- 使用 root 登录
mysql -u root -p

-- 创建数据库（如未创建）
CREATE DATABASE IF NOT EXISTS medicare
    CHARACTER SET utf8mb4
    COLLATE utf8mb4_unicode_ci;

-- 创建专用用户（推荐）
CREATE USER 'medicare'@'localhost' IDENTIFIED BY 'medicare123';
GRANT ALL PRIVILEGES ON medicare.* TO 'medicare'@'localhost';
FLUSH PRIVILEGES;
```

### 2.3 执行基线脚本

```bash
# 方式一：命令行
mysql -u root -p medicare < sql/schema_baseline.sql

# 方式二：MySQL Workbench / DataGrip 等图形工具导入
```

> 脚本会自动创建 11 张核心表 + 初始数据（5 科室、6 医生、5 药品、1 管理员）

### 2.4 验证数据库

```sql
USE medicare;
SHOW TABLES;
-- 应看到 11 张表
```

## 三、应用程序部署

### 3.1 获取安装包

```bash
# 项目根目录下执行打包（开发环境）
mvn clean package

# 生成的可执行 JAR 位于：
# target/mediCare-1.0.0-SNAPSHOT.jar
```

### 3.2 配置数据库连接

如数据库账号密码与默认不同，请修改源码后重新打包：

**文件**：`src/main/java/com/medicare/util/ConnectionConfig.java`

```java
private static final String JDBC_URL = "jdbc:mysql://localhost:3306/medicare?"
        + "useSSL=false"
        + "&serverTimezone=Asia/Shanghai"
        + "&characterEncoding=utf8mb4"
        + "&allowPublicKeyRetrieval=true";
private static final String USERNAME = "root";      // 修改为你的用户名
private static final String PASSWORD = "mysql";     // 修改为你的密码
```

### 3.3 启动应用

```bash
# 方式一：直接运行
java -jar target/mediCare-1.0.0-SNAPSHOT.jar

# 方式二：指定内存参数（推荐生产环境）
java -Xms512m -Xmx2g -jar target/mediCare-1.0.0-SNAPSHOT.jar

# 方式三：后台运行（Linux/macOS）
nohup java -jar target/mediCare-1.0.0-SNAPSHOT.jar > app.log 2>&1 &
```

### 3.4 验证启动

启动成功后：
- 日志输出在 `logs/medicare.log`
- 控制台应看到：`HikariCP 连接池初始化完成: MediCare-HikariCP`
- JavaFX 主界面自动打开

## 五、常见问题

### Q1: 启动报错 `javafx.controls` 找不到？

确保使用 **Java 17**（不是 Java 8）。JavaFX 已打包在 JAR 中，无需额外安装。

```bash
java -version
# 应显示 openjdk version "17.x.x"
```

### Q2: 数据库连接失败（时区错误）？

已在连接串中预置 `serverTimezone=Asia/Shanghai`。如仍报错，请检查 MySQL 时区：

```sql
SELECT @@global.time_zone, @@session.time_zone;
SET GLOBAL time_zone = '+8:00';
```

### Q3: 局域网其他电脑如何访问？

1. 修改 `ConnectionConfig.java` 中的 `localhost` 为数据库服务器 IP
2. MySQL 需开启远程访问：`GRANT ... TO 'user'@'%'`
3. 防火墙放行 3306 端口

### Q4: 如何备份数据？

```bash
# 备份
mysqldump -u root -p medicare > medicare_backup.sql

# 恢复
mysql -u root -p medicare < medicare_backup.sql
```
